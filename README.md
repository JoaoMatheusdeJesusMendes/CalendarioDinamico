# Projeto
O projeto se refere a um site comportando um calendário dinâmico para controle de tarefas e agendamentos. Com gameficação de processos e visualização inteligente!

## Foco
Focado em organizar de maneira dinamica a rotina do usuario, permitindo-o gameficar suas tarefas a fim de ajudar no foco e organização do seu dia a dia. Além disso, é possibilitado a vizalização por dia, semana, mês e ano a fim de verificar suas taxas de desempenho em espaços determinados de tempo.

## Ferramentas
O site utiliza o Vue, TypeScript e MongoDB.
